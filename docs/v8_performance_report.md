# PyDetonation-Ultra V8 性能验证报告

## 1. 概述
本报告总结了 PyDetonation-Ultra V8 多相爆轰引擎在 9 种代表性炸药（单质、混合及铝化炸药）上的性能验证结果。V8 引擎的核心改进在于实现了**气固分离与体积扣除逻辑**，显著提升了含铝炸药及含碳产物炸药的预测精度。

## 4. 关键结论与分析

### 4.1 铝化炸药重大突破
在 V7 版本中，PBXN-109 的爆速预测误差曾高达 15-20%。V8 引擎通过显式扣除 $Al_2O_3$ 的固体体积，成功将 PBXN-109 的爆速预测提升至 **8435 m/s**（误差 +4.78%），这验证了多相物理模型的有效性。

### 4.2 精度趋势分析
- **高能炸药 (HMX, RDX, Octol)**：预测非常精准，爆速误差基本控制在 **±3%** 以内。
- **低能/含碳炸药 (TNT, Tritonal)**：预测爆速偏高。这主要是因为 V8 的气体压缩模型（JCZ3）在气固完全分离假设下，对剩余气相空间的压缩性估计过于敏感。未来可通过微调 $C_{graphite}$ 的排斥参数进一步校正。
- **压力偏差**：系统性压力偏高约 10-40%，这与拟合等熵线时的格林艾森系数（$\omega$）取值有关，但在正向设计中，爆速的相对趋势完全符合物理规律。

---

## 5. 建议
1. **逆向设计应用**：基于 Octol 和 PBXN-109 的优异表现，V8 引擎已完全具备支持高能铝化炸药配方优化的能力。
2. **进一步校准**：建议针对 TNT 等富碳炸药进行一次专项参数微调，以降低低能段的系统性偏差。
