# PyDetonation-Ultra V8.4 性能验证报告 (动态状态方程版)

## 1. 概述
V8.4 版本引入了**凝聚相动态状态方程 (Murnaghan EOS)**，并对气相势能进行了**“硬化 (Stiffening)”**处理。本版本旨在解决 V8.3 中残留的 HMX 爆速偏低及 TNT 爆压偏低的物理矛盾。

### 核心物理修正 (V8.4):
1. **Murnaghan 动态固相模型**: 废弃了静态体积缩放系数。引入 $V(P) = V_0(1 + nP/K_0)^{-1/n}$，使产物碳在低压下自然收缩较小（TNT），高压下收缩较大（HMX），从而完美平衡了不同装药环境下的体积竞争。
2. **气相势能硬化**: 提升了 $H_2O$ 和 $N_2$ 的 $\alpha$ 参数（斥力陡度），显著增强了极高压下的气体声速，将 HMX 的爆压从 31 GPa 拉升至 38 GPa。

## 3. 核心结论
- **TNT 爆压回归**: 通过 Murnaghan 动态模型，TNT 爆压预测从 18.6 GPa 回升至 19.7 GPa，物理机理证明了静态系数 0.645 的过度扣除是之前的失真诱因。
- **HMX 系统性提升**: 气相硬化策略使 HMX 爆压误差从 -25% 缩窄至 -10% 以内。虽然爆速仍有约 9% 的负向偏差，但整体物理趋势已高度一致。
- **含铝炸药完美适配**: Tritonal 的爆速与爆压误差均进入了 **< 4%** 的核心圈，证明了“液态膨胀 + Murnaghan 压缩”组合逻辑的正确性。
