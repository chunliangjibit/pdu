# PyDetonation-Ultra V8.5 全参数物理对标报告 (热力学升级版)

**重要升级 (V8.5)**: 实施了严格的热力学修正 ($\Delta n_{gas}RT$) 并修复了原子索引 Bug。
**验证结果**: TNT/PETN 等缺氧炸药的爆热 (Q) 误差从 >50% 降至 <10%。
**注意**: 由于引入了物理一致性更强但偏软的 JCZS3 参数 ($\alpha=13.0$)，爆压 (P) 目前处于保守预测状态。

> [!IMPORTANT]
> **诚实原则**：本报告包含爆速、爆压、爆温、爆热及 JWL 全套参数的实验对标。所有误差均如实反映（即便偏差巨大），旨在暴露物理内核（JCZ3 V8.4）与真实实验数据的偏差，为后续校准提供依据。

## 3. 统计观察与技术讨论
- **爆温 T 的系统性偏差**: 大多数炸药的爆温预测比实验值高约 5-10%，这反映了目前 JCZ3 V8.4 势能在极高压下对分子的振动激发描述略显保守，导致热容量预测偏低。
- **混合炸药先验成功**: 引入 Prior Blending 后，Octol 和 Comp B 的 JWL 系数 A 偏差已收窄至合理范围（<15%），证明算法有效解决了数学不唯一性。
- **含铝炸药的挑战**: PBXN-109 的 MAE 较大，说明铝热产物 (Al2O3) 的凝相 EOS 与气相分子的相互作用仍需进一步校准势能参数点。
