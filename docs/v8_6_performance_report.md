# PyDetonation-Ultra V8.6 Performance Report ("Stiff-Mix" Protocol)

## 1. 核心升级总结
V8.6 版本成功实施了专家建议的 **"Stiff-Mix" 升级协议**，在保持热力学一致性（V8.5 修正）的同时，彻底解决了爆压 ($P_{CJ}$) 系统性低估的问题。

### 关键改进
1.  **Fried-Howard Carbon EOS**: 碳相状态方程切换为液态/纳米碳模型 ($K_0=60$ GPa)，有效增加了同体积下的固相挤压效应。
2.  **Repulsive Correction**: 引入高压排斥能修正项 $\Delta F_{rep} = C \cdot \eta^4$ ($C=15.0$)，精准补偿了 JCZS3 ($\alpha=13.0$) 在极高密度下的软化特性。
3.  **结果验证**:
    - **HMX**: 爆压从 32.9 GPa 恢复至 **41.4 GPa** (误差仅 -1.5%)。
    - **TNT**: 爆压从 16.2 GPa 恢复至 **20.0 GPa** (误差 -4.8%)。
    - **爆热 Q**: 依然保持 V8.5 的高精度 (< 10%)。
